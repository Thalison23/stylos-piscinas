# Imagens dos Produtos — Stylos Piscinas

Pasta: **`public/products/`**

Coloque cada imagem aqui com o **nome exato** indicado na tabela abaixo.

## Como funciona

1. O site lê automaticamente desta pasta (`public/products/`) — basta colocar o arquivo com o nome certo.
2. Se a imagem **não existir**, o card mostra automaticamente o ícone + gradiente azul (sem erro visual e sem quebrar o layout).
3. **Sem rebuild necessário** — atualizou a imagem, recarrega a página (Ctrl+Shift+R).
4. Pode adicionar **em lotes** (5/10/20 por vez). O site funciona normalmente enquanto a coleção é completada.

## Especificações técnicas

| Item | Valor |
|---|---|
| Formato | `.webp` (obrigatório) |
| Resolução | 600×600 (quadrado) ou 800×600 (4:3) |
| Tamanho do arquivo | 30 – 80 KB cada |
| Qualidade | 80% – 85% |
| Background | Branco ou transparente |
| Foco | Produto centralizado, sem texto sobreposto |

## Como gerar (rápido e grátis)

1. Conseguir a foto (foto do estoque ou foto oficial da marca)
2. Acessar **squoosh.app** ou **cloudconvert.com**
3. Reduzir resolução, exportar em **WebP qualidade 80%**
4. Renomear conforme tabela abaixo
5. Salvar nesta pasta

## ⚠️ Atenção sobre marcas

Várias linhas no catálogo são de marcas registradas (HTH, Genco, Pace, Sodramar, Netuno, Hidroall, Quimil etc).

- Se a Stylos tem **fotos próprias** do estoque → use elas
- Se for usar **foto oficial da marca** → confirme autorização ou pegue do site oficial do fabricante (usa pra revenda autorizada)
- Em caso de dúvida → não suba a imagem (o fallback cobre, e não tem risco jurídico)

## Como conferir o nome certo sem decorar tabela

O nome do arquivo é gerado pela função `slugify()` no JS do site (linha próxima à `imagemDoProduto`). Pra conferir um nome específico, abre o site, abre o DevTools (F12) e roda no Console:

```js
imagemDoProduto({id: '000044', nome: 'T ACAO CLORO EST.GRAN BD 10KG'})
// retorna: "products/000044-t-acao-cloro-est-gran-bd-10kg.webp"
```
